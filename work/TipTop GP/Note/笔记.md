# per和4fd的转换

    用于4fd无法编译时。
    在其他主机上编译4fd后，下载per文件到你主机上。
    通过命令 gsform -import xxxx.per 将per文件反编译为4fd文件，在讲4fd文件放到4fd目录下重新编译。
